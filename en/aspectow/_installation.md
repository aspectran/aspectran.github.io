## 1. 시스템 요구사항

Aspectow는 Java 8 이상의 버전이 설치된 다음과 같은 운영체제 상에서 실행될 수 있습니다.

* Windows
* Unix/Linux
* Mac OS

## 2. 다운로드

Aspectow는 별도의 설치파일을 제공하지 않으며, GitHub의 저장소로부터 직접 다운로드 할 수 있습니다.
